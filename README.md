# Service Booking Platform

A modern service booking marketplace built with Django.

## Features

- Buyer & Seller Login
- Service Booking
- Seller Dashboard
- Buyer Dashboard
- Booking Status System

## Technologies Used

- Python
- Django
- Bootstrap 5
- SQLite

## Installation

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py runserver
